# Goalie Gear Calculator

A professional goalie equipment sizing calculator for [Goalie Coaches](https://goaliecoaches.com). This tool helps goalies of all ages and skill levels generate accurate gear sizing recommendations based on their body measurements.

## Overview

The Goalie Gear Calculator is a web-based application that takes user measurements and calculates appropriate sizing for all essential goalie equipment. It provides brand-specific sizing recommendations and links to comprehensive buying guides to help goalies make informed purchasing decisions.

## Features

### Core Functionality

- **Comprehensive Measurement Input**: Supports multiple body measurements including:
  - Height and weight
  - Head circumference
  - Hand length and width
  - Arm span
  - Floor-to-knee (FTK) and ankle-to-knee (ATK) measurements
  - Neck circumference
  - Waist size
  - Shoe size and skate fit preferences

- **Dual Unit System**: 
  - Imperial (inches, lbs)
  - Metric (cm, kg)
  - Automatic conversion and dual-unit display

- **Age-Based Categorization**:
  - Youth (ages 3-8)
  - Junior (ages 7-13)
  - Intermediate (ages 12-14)
  - Senior (ages 15+)

### Equipment Sizing

The calculator provides sizing recommendations for:

1. **Helmet (Mask)** - Brand-specific sizing for Bauer, CCM, Warrior, and Vaughn
2. **Neck Guard** - Size recommendations with fit guidelines
3. **Chest Protector** - Height and arm span-based sizing with brand-specific recommendations
4. **Glove (Catcher)** - Hand length and width-based sizing
5. **Blocker** - Matched to glove sizing
6. **Goalie Pants** - Height and body type-based sizing
7. **Jock/Jill** - Waist-based protective cup sizing
8. **Leg Pads** - ATK-based sizing with thigh rise recommendations
9. **Skates** - Shoe size conversion with fit system recommendations
10. **Stick** - Height-based paddle and overall length recommendations

### Additional Features

- **Brand-Specific Recommendations**: Detailed sizing charts for major brands (Bauer, CCM, Warrior, Vaughn, Brian's)
- **Measurement Guide**: Built-in instructions for taking accurate measurements
- **Buying Guide Links**: Direct links to comprehensive buying guides on goaliecoaches.com for each equipment category
- **PDF Export**: Download sizing results as a formatted PDF for reference
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Print-Friendly**: Optimized print styles for physical reference

## Usage

1. Open `index.html` in a web browser
2. Select your preferred unit system (Imperial or Metric)
3. Enter your measurements in the form
4. Click "Calculate Sizes" to generate recommendations
5. Review sizing results with brand-specific recommendations
6. Click on buying guide links for detailed product information
7. Download results as PDF for future reference

## Measurement Tips

- **Head Circumference**: Measure 1/2" above eyebrows and ears at widest part of head
- **Hand Length**: Measure from base of palm (where wrist meets hand) to tip of middle finger
- **Floor to Knee**: Stand straight and measure from floor to center of kneecap
- **Neck Circumference**: Measure around neck just below Adam's apple, leaving room for comfort
